Project Features

  • Responsive design for all devices (mobile, tablet, desktop)
  • Mobile-first approach with progressive enhancement
  • Single HTML file containing all pages and functionality
  • Light and dark mode with user preference persistence
  • Semantic HTML5 markup for accessibility
  • Interactive elements with feedback and validation
  • Custom styling with CSS variables for easy theming
  • Smooth animations and transitions for better UX

Technology Stack

  • HTML5 for semantic structure
  • CSS3 with Flexbox/Grid for responsive layouts
  • Vanilla JavaScript for interactivity
  • CSS Custom Properties (variables) for theming
  • Media queries for responsive breakpoints
  • Font Awesome icons for visual elements
  • LocalStorage for theme preference persistence
